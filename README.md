# 🚀 Kişisel Web Projesi
Bu proje, modern web teknolojileri kullanılarak geliştirilmiş, çok dilli destek sunan ve kullanıcı dostu bir arayüze sahip kişisel bir web sitesidir.

---

# 📂 Proje Yapısı
Klasördeki dosyaların işlevleri şunlardır:

 Sitenin ana sayfası ve giriş noktası.
 Hakkımda bölümünün yer aldığı detaylı sayfa.
 Kullanıcıların iletişim kurabileceği form ve bilgilerin olduğu sayfa.
 Sitenin tüm görsel tasarımını ve düzenini (layout) kontrol eden stil dosyası.
 Sitenin dinamik özelliklerini ve etkileşimlerini yöneten ana JavaScript dosyası.
 Sitedeki çoklu dil desteğini (Türkçe/İngilizce vb.) sağlayan veri dosyası.

## 🛠️ Kullanılan Teknolojiler
HTML5: Yapısal kurgu için.

CSS3: Modern ve responsive (mobil uyumlu) tasarım için.

JavaScript (ES6+): Dinamik içerik yönetimi ve dil seçeneği için.
